Features and About

The Task

Caff3.ine was made as part of a group assignment for the Game Design and Development Module.

Created in a team of 7 people, with myself working on gameplay programming, technical writing and cutscenes.

Features

-A game about running a coffee shop in a cyberpunk authoritarian society, where every citizen lives in fear of "The Ministry".

-This game had plenty of features to discuss. These include,

-A fully working dialogue system using JSON to store data types regarding characters and their dialogue paths.

-Scripted movement of NPCs and an accompanying waypoint system to control them for scenes.

-A uniquely made Scene creation tool which utilises the waypoint movement system, the JSON dialogue system and the Cinemachine software to create a smooth running of the scene.

-Multiple different endings and paths the player can take, with routes that can change dialogue based on player choice.

-A coffee creation and design system and accompanying UI.

-Much More

Inspirations

Two key inspirations for this game were the two games "Cyberpunk 2077" and "Paper's Please!". Additionally, the book "1984" was a key narrative inspiration for the game.

The aesthetic was designed to resemble a more colorful version of the world of Cyberpunk 2077, with large monitors and holograms, but also a robotic coffee barista who is there to take orders.

The gameplay was designed in a similar way as Paper's Please, where the player is greeted regularly by NPC living in the world, and it's through these NPCs that the player can learn about the world further.

The narrative of the "Ministry" takes heavy inspiration from 1984, in a similar way to "The Party" that is in control of the entire nation, and changes the narrative as they see fit.
