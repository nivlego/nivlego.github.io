Features and About

The Task

Snip and Shear was a group submission for Global Game Jam 2023, where the theme was "Roots".

We worked in a team of 6 people, with myself working on programming the movement of enemies and of the player, programming of items and also working together on game design.

Features

When working on this project, the most important choice was to create a game which was enjoyable to play, and was reminiscent of it's inspirations.

I spent the time coding the movement of enemies and the player, and worked on creating a system to pick up and use items or weapons.

This game was made in a shorter time period than the previous Game Jam.

Inspirations

Two key inspirations for this game was classic "Doom" game, and also originally the game "Office simulator", but this quickly changed to a more generic retro style experience.

The level design took inspiration from doom, but additionally so did the gun shooting, enemies themselves and in particular, the animation of the weapons.
